# STX/stSTX 내재가치 기준 기회 횟수 (샘플 7일)

작성일: 2026-02-19

## 결론

- 가능하다.  
- 내재가치(`intrinsic_stx_per_ststx`) 대비 시장비율(`market_stx_per_ststx`)의 괴리율 `edge_pct`를 기준으로 기회 횟수를 집계할 수 있다.

## 집계 기준

- `edge_pct = (market / intrinsic - 1) * 100`
- 임계치별 카운트:
  - `|edge_pct| >= 0.25%`
  - `|edge_pct| >= 0.50%`
  - `|edge_pct| >= 1.00%`

## 샘플 결과 (최근 7일)

- `|edge| >= 0.25%`: `6/7`
- `|edge| >= 0.50%`: `5/7`
- `|edge| >= 1.00%`: `3/7`

상세 샘플:
- 2026-02-14: `+0.2593%`
- 2026-02-15: `+0.5699%`
- 2026-02-16: `-1.6000%`
- 2026-02-17: `-1.2739%`
- 2026-02-18: `-1.2703%`
- 2026-02-19: `-0.5512%`
- 2026-02-19: `-0.0970%`

## 주의사항

- 이 값은 1차 스크리닝(후보 탐색)용이다.
- 실제 아비트라지 가능 여부는 DEX 체결가격, 유동성 깊이, 슬리피지, 수수료를 추가 반영해야 확정 가능하다.
- 30일 전체를 동일 방식으로 집계하려 했지만 Hiro API rate-limit(HTTP 429)로 한 번에 끝내기 어려웠다.

## 다음 확장

- 호출 간격/캐시/배치 저장을 넣어 30일, 90일 기회 횟수 표를 같은 방식으로 확장 가능.
