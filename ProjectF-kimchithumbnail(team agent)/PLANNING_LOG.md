# ProjectF - 김치 썸네일 메이커 기획 로그

## 프로젝트 개요
**목표**: 한국인 관심 폴리마켓 주제 선별 + 자동 썸네일 생성

## 2026-02-09 기획 세션

### 사용자 아이디어 요약

#### 에이전트 1: 한국인 관심 주제 큐레이터
- 폴리마켓 전체를 모니터링하면서 조건에 맞는 주제 선별
- 조건: 거래량 일정 이상 + breaking + 뜨거운 주제 → "한국인이 재밌어할 것" 판단
- 관리자 수동픽 보조 또는 자동 추천

#### 에이전트 2: 썸네일 디자이너
- 브랜드 통일성 유지하며 썸네일 생성
- 실제 인물/이슈 이미지를 활용한 합성 작업

---

## Claude 조언

### 1. 구현 가능성 분석

#### 에이전트 1 (큐레이터) - 가능
- Claude/GPT가 충분히 잘 수행 가능
- 폴리마켓 API에서 마켓 목록 조회 → AI가 한국 관련성 판단
- 기존 ProjectC의 알파 탐지 로직 재활용 가능

**필요 기능:**
- 한국 관련 키워드 필터링 (K-pop, 삼성, 한국 정치인 등)
- 한국인 관심도 점수화 (트렌드, 문화적 맥락)
- 한국어 번역/요약

#### 에이전트 2 (썸네일) - 부분적으로 가능, 주의 필요

**Claude의 한계:**
- Claude는 이미지를 **생성할 수 없음** (읽기만 가능)
- 직접 포토샵/합성 작업 불가

**대안 솔루션:**

| 방식 | 장점 | 단점 |
|------|------|------|
| **DALL-E 3 API** | 고품질, 빠름 | 실제 인물 생성 제한, 브랜드 일관성 어려움 |
| **Stable Diffusion API** | 커스텀 가능, 저렴 | 셋업 필요, 품질 변동 |
| **HTML 템플릿 + Puppeteer 스크린샷** | 브랜드 일관성 최고, 정확한 레이아웃 | 실제 사진 합성은 별도 처리 필요 |
| **Canva API** | 사용 편리, 템플릿 풍부 | 유료, 자동화 제한 |

### 이미지 생성 API 서비스 조사 결과 (2026-02-09)

#### 템플릿 기반 이미지 생성 API 비교표

| 서비스 | 시작 가격 | 크레딧/이미지 | 무료 플랜 | 특징 |
|--------|----------|--------------|----------|------|
| **[Bannerbear](https://www.bannerbear.com/)** | $49/월 | 1,000개 | 30개 무료 | 가장 안정적, 비디오도 지원 |
| **[Placid.app](https://placid.app/)** | $19/월 | 500개 | 미확인 | No-code 친화적, Zapier 통합 |
| **[Templated.io](https://templated.io/)** | $19/월 | 500개 | 50개 무료 | AI 템플릿 생성 기능 |
| **[RenderForm](https://renderform.io/)** | $9/월 | 250개 | 50개 무료 | 가장 저렴, Pay-as-you-go 가능 |
| **[DynaPictures](https://dynapictures.com/)** | €39/월 | 변동 | 있음 | 이미지당 $0.003~$0.039 |
| **[APITemplate.io](https://apitemplate.io/)** | 미확인 | 50개/월 무료 | 있음 | 드래그앤드롭 편집기 |

#### 작동 방식 (공통)

```
1. 웹 에디터에서 템플릿 디자인 (드래그앤드롭)
   - 텍스트 레이어, 이미지 레이어, 도형 등 배치
   - 동적으로 바꿀 부분 지정 (예: {{title}}, {{image_url}})

2. API 호출로 이미지 생성
   POST /render
   {
     "template_id": "abc123",
     "modifications": {
       "title": "트럼프 vs 바이든",
       "image_url": "https://example.com/trump.jpg"
     }
   }

3. 생성된 이미지 URL 반환 (PNG/JPEG)
```

#### 추천 옵션

|------|-----------|
| **MVP/테스트** | RenderForm (가장 저렴, 50개 무료) |

| **비용 최적화** | DynaPictures (이미지당 과금) |

### 3. 실제 인물 이미지 관련

> **사용자 피드백**: 폴리마켓에 올라오는 인물들은 대부분 공인이고, 인물 사진 크롭해서 활용할 것이므로 크게 걱정 안 해도 됨

#### 이미지 수집 방법
- 구글 이미지 검색 → 인물 사진 크롭
- 뉴스 기사에서 이미지 추출
- 위키피디아/공식 프로필 사진
