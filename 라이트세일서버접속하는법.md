# 라이트세일 서버 접속하는 법

## 1) 서버 IP 확인
- AWS Lightsail 콘솔에서 인스턴스 카드에 표시되는 IPv4 주소를 확인한다.
- 현재 사용 중인 서버 IP: `43.203.122.192`

## 2) SSH로 접속
- 터미널에서 아래 명령어로 접속한다.

```bash
ssh ubuntu@43.203.122.192
```

## 3) 접속이 안 될 때 체크
- Lightsail 콘솔에서 인스턴스 상태가 `Running`인지 확인
- 네트워킹에서 `22/tcp` (SSH) 포트가 열려 있는지 확인
- 처음 접속 시 `yes`를 입력해 호스트 키 등록

## 4) 서버 종료/재부팅
- 콘솔에서 Stop/Reboot로 제어 가능
