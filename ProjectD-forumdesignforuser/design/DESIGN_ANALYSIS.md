# 디자인 분석

MarketMotion 커뮤니티 피그마 시안 분석 결과

---

## 핵심 컨셉 3가지

### 1. 마켓 중심 커뮤니티
- 모든 글이 특정 마켓(종목/자산)과 연결
- 글 작성 시 관련 마켓 선택 필수/선택 가능
- 마켓별로 관련 글 모아보기 지원

### 2. 예측 기반 신뢰 시스템
- 강세(Bullish) / 약세(Bearish) 의견 표명
- 예측 적중률 자동 추적
- 적중률 기반 배지 및 신뢰도 부여

### 3. 투명한 유저 프로필
- 포트폴리오 공개 옵션
- 과거 예측 기록 열람 가능
- 레벨/배지로 신뢰도 시각화

---

## UI/UX 특징

### 전체 스타일
- **다크 모드 기반**: 투자 플랫폼 특성 반영
- **카드형 레이아웃**: 글 목록을 카드로 표시
- **컬러 코딩**: 강세(초록), 약세(빨강) 직관적 구분

### 네비게이션
- 상단 헤더: 로고, 검색, 알림, 프로필
- 사이드바: 홈, 인기, 팔로잉, 마켓, 설정
- 하단 탭 (모바일): 핵심 메뉴 빠른 접근

### 반응형 디자인
- 데스크톱: 3컬럼 레이아웃 (사이드바 + 피드 + 위젯)
- 태블릿: 2컬럼 레이아웃
- 모바일: 1컬럼 + 하단 탭

---

## 화면별 구성 요소

### 홈 피드
- 추천/최신/팔로잉 탭
- 글 카드: 작성자, 마켓 태그, 강세/약세, 좋아요, 댓글
- 인기 마켓 위젯
- 추천 유저 위젯

### 글 상세
- 본문 내용
- 연결된 마켓 정보 (실시간 가격)
- 작성자 프로필 카드
- 댓글 섹션
- 관련 글 추천

### 글 작성
- 리치 텍스트 에디터
- 마켓 검색 및 연결
- 강세/약세 선택
- 이미지/차트 첨부
- 미리보기

### 프로필
- 기본 정보 (닉네임, 배지, 레벨)
- 적중률 통계
- 포트폴리오 (공개 시)
- 작성 글 목록
- 팔로워/팔로잉

### 마켓 페이지
- 마켓 기본 정보
- 실시간 가격 차트
- 커뮤니티 감성 지표 (강세 vs 약세 비율)
- 관련 글 목록

## 토스와의 비교

### 유사점
- 깔끔한 다크 모드 UI
- 직관적인 정보 계층
- 부드러운 애니메이션
- 원형 프로필 이미지

### 차별점
| 요소 | 토스 | MarketMotion |
|------|------|--------------|
| 목적 | 금융 서비스 | 투자 커뮤니티 |
| 핵심 기능 | 송금, 투자 | 토론, 예측 |
| 컨텐츠 | 개인 금융 | 유저 생성 컨텐츠 |
| 신뢰 지표 | 없음 | 적중률, 배지 |

### 벤치마킹 포인트
- 토스의 직관적 정보 표시 방식
- 스와이프 인터랙션
- 바텀시트 활용
- 로딩 스켈레톤 UI
