# MM.pro 게시판 정책 정리

## 개요

이 문서는 MM.pro 커뮤니티 게시판의 정책을 정리한 팀 공유용 문서입니다.

**작성 배경:** 디시인사이드 + 토스증권 커뮤니티를 벤치마킹하여, Polymarket 한글화 플랫폼에 맞는 게시판 정책을 설계했습니다.

**핵심 철학:**
- 게시판이 메인, 예측은 부가 기능
- 정치 토론은 커뮤니티의 핵심 → 표현의 자유 최대한 보장
- 맞춘 거 자랑하는 재미 + 틀려도 기분 안 나쁘게

---

## 1. 피드 구조

### 3탭 구조

| 탭 | 용도 | 상단 인기글 |
|----|------|:----------:|
| **최신글** | 일상적 소비 | 2개 |
| **주제별** | 카테고리별 탐색 | 2개 |
| **뜨거운 반응** | 인기글 전용 | - |

### 왜 3탭인가?
- 처음엔 2탭(최신글/주제별)으로 시작
- "인기글 상단 고정이 스크롤 피로를 줄까?" 고민
- 뜨거운 반응 탭을 분리하여 인기글 전용 공간 제공
- 각 탭 상단 인기글은 2개로 통일 (스크롤 피로 최소화)

### 스크롤 방식
- **무한스크롤** (세로만)
- 가로 스크롤 없음 (UX 혼란 방지)
- 투자 커뮤니티 특성상 실시간성 + 모바일 중심

→ 대부분 클릭 1번 이하로 해결

## 3. 인기글 알고리즘

### 점수 공식

```
인기 점수 = (기본 점수 + 보너스 점수) × 시간 감쇠율
```

### 기본 점수
- 좋아요: `10 × log₂(좋아요 수 + 1)` (로그 스케일)
- 댓글: 1개당 +5점
- 조회수: 100회당 +1점 (최대 +20)

### 보너스 점수
- 예측 연동 글: +20점
- 작성자 적중률 60%+: +15점
- 작성자 적중률 70%+: +30점
- 작성자 Lv.4 이상: +10점
- 이미지/차트 포함: +5점

### 시간 감쇠율
```
감쇠율 = 1 / (1 + 경과시간(h) / 6)^1.5
```

### 인기글 진입 기준
- 좋아요 15개 이상 + 댓글 5개 이상
- 또는 1시간 내 좋아요 10개 이상 (급상승)
- 익명 글도 인기글 진입 **가능**

## 4. 익명 정책

### 익명 글 vs 익명 댓글

| 항목 | 익명 글 | 익명 댓글 |
|------|:------:|:--------:|
| 작성 위치 | 마켓 페이지만 | 어디서든 |
| 피드 노출 | ✅ | - |
| 좋아요 받기 | ✅ | ✅ |
| 좋아요 주기 | ❌ | ✅ |
| 인기글 진입 | ✅ | - |
| 쿨타임 | 3분 | 10초 |

### 익명 닉네임 규칙
- 랜덤 닉네임 부여 (예: 졸린판다, 행복한고양이)
- **같은 글 내에서 동일인 = 같은 닉네임 유지**
- 다른 글에서는 새로운 닉네임

## 5. 댓글 시스템

| 항목 | 정책 |
|------|------|
| 답글 구조 | 1단만 (대댓글 X) |
| 정렬 | 시간순 (오래된 순) |
| 글자 제한 | 1,000자 |
| 수정/삭제 | 10분 이내만 |
| 삭제 시 | 답글 있으면 "삭제된 댓글입니다" 표시 |

---

## 6. 글 수정/삭제

| 항목 | 글 | 댓글 |
|------|:--:|:----:|
| 수정 가능 시간 | 10분 | 10분 |
| 삭제 가능 시간 | **무제한** | 10분 |
| 수정 이력 | "(수정됨 N분 전)" 표시 | 동일 |

**설계 의도:**
- 글 삭제는 본인 콘텐츠 관리 권한 존중
- 댓글 삭제 제한은 토론 맥락 보존

## 7. 마켓 종료 처리

### 예측 결과 표시

**원칙:** "적중/실패" 같은 개인 평가 ❌ → 객관적 사실만 ✅

```
🟢 강세 예측 · 결과: $102k 돌파 ✓
🔴 약세 예측 · 결과: $102k 돌파 ✓  ← 틀렸지만 중립적
```

### 노출 정책
- 마켓 종료 후 24시간: 정상 노출 (자랑 타임)
- 24시간 이후: 점진적 노출 감소

### 수정/댓글
- 예측 수정: **마켓 종료 전까지만** (종료 후 수정 불가)
- 댓글: **언제든 가능**

---

## 8. 알림 시스템

### 알림 종류
- 내 글에 댓글 / 내 댓글에 답글
- 좋아요 (**묶음**: "A님 외 5명이 좋아요")
- 새 팔로워 / 팔로잉 유저 새 글
- 예측 결과 확정
- 인기글 진입 / 배지·레벨업

### 설정
- 전체 ON/OFF만 (종류별 설정 X)
- 30일 보관 후 자동 삭제
- 웹 푸시 지원

---

## 9. 검색/북마크

### 검색
- 범위: 글 제목+본문 + 유저 닉네임 (댓글 제외)
- 필터: 주제, 기간, 작성자 레벨, 예측 방향
- 정렬: 최신순 (기본) / 관련도순

### 북마크
- 대상: 글 + 댓글 + 유저
- 폴더 분류: 없음 (단순 목록)
- 삭제된 글: "삭제된 글입니다" 표시 유지

---

## 10. 신고/차단

### 신고
- 대상: 글 + 댓글
- 사유: 스팸/광고, 욕설/비하, 허위 정보, 도배, 기타
- 처리: **5건 이상 → 자동 숨김 + 모더레이터 알림**
- 결과 알림: 처리 완료 시 신고자에게 알림

### 차단
- 효과: 상대 글/댓글 안 보임
- 알림: 없음 (몰래 차단)
- 해제: 언제든 가능

## 11. 키워드 필터

| 카테고리 | 처리 |
|---------|------|
| 일반 욕설 | 마스킹 (시**) |
| 경미한 표현 | **허용** |
| 성적/비하/가족 비하 | 마스킹 |
| 인종 차별 | 자동 숨김 + 신고 |
| **정치 비하** | **허용** (커뮤니티 핵심) |
| 스팸/광고/도박 | 자동 숨김 |

**핵심:** 정치 토론은 커뮤니티 핵심 → 정치 비하 필터 최소화

## 13. 프로필/계정

### 프로필 설정
- 닉네임: 2~15자, 90일에 1회 변경, 중복 불허
- 프로필 이미지: 커스텀 업로드 가능
- 자기소개: 100자 제한

### 팔로우
- 알림: 있음
- 목록: 본인만 확인 가능
- 맞팔 표시: 안 함

### 회원가입
- Privy 사용 (지갑/소셜/이메일)

### 탈퇴
- 유예 기간: 7일 (철회 가능)
- 글/댓글: 유지 (닉네임 → "탈퇴한 사용자")
- 재가입: 30일 제한

## 15. 서비스 구조

### 메인 vs 커뮤니티

```
┌─────────────────────────────────┐
│  메인 페이지 (예측 시장)         │
│  - Polymarket 마켓 한글화       │
│  - 마켓별 댓글 (디시 스타일)     │  ← 로그인 없이 작성 가능
└──────────────┬──────────────────┘
               │ 자동 연동
               ▼
┌─────────────────────────────────┐
│  커뮤니티 탭 (피드)              │
│  - [최신글] [주제별] [뜨거운 반응] │  ← 로그인 필요
└─────────────────────────────────┘
```

### 글 작성 연동
- 마켓 페이지 글 → 커뮤니티 피드 자동 노출
- 마켓 정보 (카테고리, 종목) 자동 상속

## 부록: 주요 결정 근거

### 왜 무한스크롤인가?
- 투자 커뮤니티는 실시간성이 중요
- 모바일 사용자 비중 높음
- 페이지네이션은 탐색엔 좋지만 참여 유도엔 무한스크롤이 유리

### 왜 인기글 2개인가?
- 3개는 스크롤 피로
- 1개는 "인기글" 느낌 약함
- 2개 + 새로고침마다 교체가 최적

### 왜 정치 비하를 허용하나?
- MM.pro는 예측 시장 커뮤니티
- 정치 토론이 핵심 콘텐츠
- 지지 정당 간 논쟁은 자연스러운 것

### 왜 적중/실패 대신 객관적 표시인가?
- "실패" 라벨은 유저 기분 상하게 함
- 예측 방향 + 실제 결과만 보여주면 충분
- 맞췄는지 틀렸는지는 보는 사람이 판단
