# 모더레이션 정책

커뮤니티 운영 및 제재 기준

---

## 모더레이션 대상

### 1. 스팸
- 동일/유사 내용 반복 게시
- 광고성 링크 무분별 공유
- 봇을 이용한 자동 게시
- 무의미한 짧은 댓글 도배

### 2. 욕설 및 비방
- 직접적 욕설 사용
- 특정 유저 지속적 비방
- 조롱/비아냥 목적의 댓글
- 개인정보 유출 (신상털기)

### 3. 허위 정보
- 확인되지 않은 루머 유포
- 의도적 오정보 전달
- 가짜 뉴스 공유
- 조작된 데이터/차트 게시

### 4. 시장 조작 시도
- 특정 종목 조직적 펌핑
- 공포 조성을 통한 덤핑 유도
- 허위 매매 인증
- 리딩방 홍보

### 5. 기타
- 저작권 침해
- 성인/폭력 컨텐츠
- 정치/종교 분쟁 유발
- 플랫폼 악용 (버그 익스플로잇 등)

---

## 제재 단계

### Level 1: 경고
**대상**: 경미한 위반 첫 발생
- 해당 컨텐츠 숨김 처리
- 경고 알림 발송
- 위반 사유 안내

### Level 2: 일시 정지 (7일)
**대상**: 경고 누적 또는 중간 수준 위반
- 글 작성/댓글 금지
- 좋아요/팔로우 금지
- 읽기만 가능

### Level 3: 장기 정지 (30일)
**대상**: 일시 정지 후 재위반 또는 심각한 위반
- 모든 활동 금지
- 프로필 비공개 전환
- 복귀 시 서약 필요

### Level 4: 영구 정지
**대상**: 장기 정지 후 재위반 또는 매우 심각한 위반
- 계정 영구 비활성화
- 동일 기기/IP 재가입 제한
- 법적 조치 검토 (필요시)

## 위반 유형별 제재 기준

| 위반 유형 | 1회 | 2회 | 3회 |
|-----------|-----|-----|-----|
| 스팸 | 경고 | 7일 정지 | 영구 정지 |
| 욕설 | 경고 | 7일 정지 | 30일 정지 |
| 허위 정보 | 경고 | 7일 정지 | 30일 정지 |
| 펌핑/덤핑 | 7일 정지 | 30일 정지 | 영구 정지 |
| 사기 | 영구 정지 | - | - |
| 혐오 발언 | 영구 정지 | - | - |
| 불법 행위 | 영구 정지 + 신고 | - | - |

## 신고 시스템

### 신고 방법
1. 해당 글/댓글의 ⋮ 메뉴 클릭
2. "신고하기" 선택
3. 신고 사유 선택
4. 추가 설명 입력 (선택)
5. 제출

### 신고 사유 옵션
- 스팸/광고
- 욕설/비방
- 허위 정보
- 시장 조작
- 저작권 침해
- 기타 (직접 입력)

### 신고 처리 과정
1. 신고 접수 (자동 알림)
2. 모더레이터 검토 (24시간 내)
3. 조치 결정
4. 신고자/피신고자에게 결과 통보

### 허위 신고
- 악의적 허위 신고 시 신고자 제재
- 신고 기능 일시 제한

## 모더레이터

### 역할
- 신고 검토 및 처리
- 커뮤니티 모니터링
- 제재 집행
- 유저 문의 대응

### 권한
- 글/댓글 숨김 및 삭제
- 유저 경고 및 정지
- 신고 처리
- 이의제기 검토

### 의무
- 공정한 판단
- 신속한 처리
- 개인정보 보호
- 월간 활동 보고

### 모더레이터 선발
- 활동 6개월 이상
- 레벨 "시니어" 이상
- 제재 이력 없음
- 지원 및 심사 통과

## 이의제기

### 이의제기 대상
- 부당하다고 느끼는 제재
- 오해로 인한 처분
- 맥락이 고려되지 않은 판단

### 이의제기 방법
1. 설정 > 계정 > 제재 내역
2. 해당 제재 선택
3. "이의제기" 버튼 클릭
4. 사유 상세 작성
5. 증거 자료 첨부 (선택)

### 처리 과정
1. 이의제기 접수
2. 다른 모더레이터 재검토 (48시간 내)
3. 필요시 추가 자료 요청
4. 최종 결정 통보

### 제한 사항
- 동일 제재에 1회만 가능
- 영구 정지는 운영팀 검토
- 결정에 불복 시 고객센터 문의

## 자동 모더레이션

### 자동 필터링
- 금칙어 자동 감지
- 스팸 패턴 탐지
- 도배 방지 (쿨다운)
- 외부 링크 검사

### AI 보조
- 혐오 발언 탐지
- 허위 정보 의심 표시
- 이상 활동 패턴 감지
- 모더레이터 우선순위 제안

### 한계
- 자동 시스템은 보조 역할
- 최종 판단은 사람이 수행
- 오탐지 시 이의제기 가능
