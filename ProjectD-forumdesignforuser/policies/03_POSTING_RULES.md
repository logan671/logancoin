# 글쓰기 규칙

MarketMotion에서 글 작성 시 지켜야 할 규칙

---

## 글 작성 형식

### 제목
- 최소 3자, 최대 100자
- 핵심 내용을 담을 것
- 낚시성 제목 지양
- 종목명 + 의견 형식 권장

**좋은 예시**
- "삼성전자 4분기 실적 분석 - 반도체 회복 기대"
- "비트코인 주요 지지선 분석"

**나쁜 예시**
- "이거 보고도 안 사면 바보"
- "급등 예고!!! 놓치면 후회"

### 본문
- 최소 50자 권장
- 마크다운 형식 지원
- 이미지/차트 첨부 가능
- 외부 링크 허용 (검증된 출처)

### 필수 포함 요소 (분석글)
- 분석 대상 명시
- 분석 근거/논리
- 리스크 요인
- 면책 문구

---

## 마켓 연동 규칙

### 마켓 연결이란
글 작성 시 관련된 종목/자산을 태그하는 기능

### 연결 방법
1. 글 작성 화면에서 "마켓 추가" 클릭
2. 종목명 또는 티커 검색
3. 원하는 마켓 선택
4. 최대 5개까지 연결 가능

### 필수 vs 선택

| 글 유형 | 마켓 연결 | 비고 |
|---------|-----------|------|
| 종목 분석 | **필수** | 분석 대상 종목 |
| 시장 전망 | 선택 | 관련 지수 등 |
| 포트폴리오 공유 | **필수** | 보유 종목들 |
| 일반 토론 | 선택 | 관련 있으면 추가 |
| 질문/초보 | 선택 | - |

### 마켓 연결 시 표시
- 글 상단에 마켓 태그 표시
- 해당 마켓의 실시간 가격
- 작성 시점 대비 변동률
- 마켓 페이지로 링크

---

## 강세/약세 의견 표시

### 포지션 선택
글 작성 시 해당 마켓에 대한 의견 표명

| 선택 | 의미 | 표시 |
|------|------|------|
| 🟢 강세 (Bullish) | 상승 예상 | 초록색 태그 |
| 🔴 약세 (Bearish) | 하락 예상 | 빨간색 태그 |
| ⚪ 중립 | 관망/분석만 | 회색 태그 |

### 적중률 계산
- 글 작성 후 7일/30일/90일 기준
- 마켓 가격 변동과 의견 비교
- 프로필에 적중률 표시

### 포지션 변경
- 기존 글의 포지션은 변경 불가
- 의견 변경 시 새 글 작성
- 변경 이력은 프로필에 기록

## 포트폴리오 인증

### 인증 방법
1. **스크린샷 인증**
   - 증권사 앱 캡처
   - 개인정보 가리기 필수
   - 날짜/시간 포함

2. **API 연동** (예정)
   - 증권사 OpenAPI 연결
   - 자동 동기화
   - 높은 신뢰도

### 인증 시 주의사항
- 타인 포트폴리오 도용 금지
- 조작된 이미지 금지
- 인증 뱃지 허위 표시 금지

### 공개 범위 설정
- 전체 공개: 누구나 열람
- 팔로워 공개: 팔로워만 열람
- 비공개: 본인만 열람

## 금지되는 글 유형

### 즉시 삭제
- **리딩방 홍보**: 유료 리딩 서비스 광고
- **사기성 글**: 수익 보장, 원금 보장 주장
- **불법 정보**: 미공개 정보, 내부자 거래
- **펌핑 글**: 조직적 매수 유도

### 경고 후 삭제
- **도배**: 동일/유사 내용 반복
- **낚시성**: 제목과 내용 불일치
- **무성의**: 내용 없는 단순 의견
- **출처 미표기**: 타인 분석 무단 전재

### 수정 요청
- **면책 누락**: 투자 권유 오해 소지
- **과도한 확신**: "100% 오른다" 등
- **마켓 미연결**: 분석글인데 종목 미태그

---

## 글 수정 및 삭제

### 수정
- 작성 후 24시간 내 무제한
- 24시간 후 수정 시 "수정됨" 표시
- 포지션(강세/약세)은 수정 불가

### 삭제
- 본인 글 삭제 가능
- 삭제해도 적중률 계산에 포함
- 댓글 있는 글 삭제 시 "삭제된 글" 표시

### 보존
- 예측 글은 90일간 삭제 제한
- 적중률 계산 완료 후 삭제 가능
- 삭제 요청은 고객센터 문의

## 댓글 규칙

### 허용
- 글에 대한 의견/질문
- 추가 정보 제공
- 건설적 비판
- 오류 지적

### 금지
- 인신공격
- 도배
- 광고/홍보
- 분쟁 조장

### 대댓글
- 2단계까지 지원
- 멘션(@) 기능 제공
- 글쓴이 답글 하이라이트
