## 배지 종류

### 🎯 예측가 배지

난이도 가중치 기반 배지 (단순 적중률 X)

| 배지 | 조건 | 아이콘 |
|------|------|--------|
| 예측가 브론즈 | 가중 점수 100점 이상 | 🎯 |
| 예측가 실버 | 가중 점수 300점 이상 | 🎯🎯 |
| 예측가 골드 | 가중 점수 700점 이상 | 🎯🎯🎯 |
| 예측가 다이아 | 가중 점수 1,500점 이상 | 💎 |

**난이도 가중치 시스템**
- 커뮤니티 컨센서스 반대로 맞추면 높은 점수
- 90% 동의 방향 적중 → **+5점** (쉬운 예측)
- 70% 동의 방향 적중 → **+10점**
- 50% 동의 방향 적중 → **+20점** (반반)
- 30% 동의 방향 적중 → **+40점** (역행 성공)
- 10% 동의 방향 적중 → **+80점** (극역행 성공)
- 틀리면 0점 (감점 없음)

**연속 적중 보너스**
- 3연속 적중: 🔥 +50 보너스
- 5연속 적중: 🔥🔥 +150 보너스
- 10연속 적중: 🔥🔥🔥 +500 보너스

### ✅ 인증 배지

신원/보유 인증 배지

| 배지 | 조건 | 아이콘 |
|------|------|--------|
| 스킨 인 더 게임 | 언급 종목 실제 보유 중 | ✅ |
| 자산 구간 표시 | 선택적 자산 구간 공개 | 💰 |
| 프로 투자자 | 전문 투자자 인증 | 🏆 |

**스킨 인 더 게임 (핵심 배지)**
- 글에서 언급한 종목을 **실제로 보유 중**인지 표시
- Privy 지갑 연동으로 자동 확인
- 보유 수량/금액은 비공개, **보유 여부만** 표시
- "이 사람은 본인이 말하는 거 실제로 들고 있음" ✅

**자산 구간 표시 (선택적)**
- 기본값: OFF (표시 안 함)
- 본인이 원할 때만 ON
- 정확한 금액 X → 구간만 표시
  - 💰 1천만+ / 💎 1억+ / 👑 10억+
- 언제든 켜고 끄기 가능

**익명 글쓰기 모드**
- 자산 배지 있어도 글 작성 시 "익명으로 작성" 옵션
- 익명 글에는 모든 자산/인증 배지 숨김
- 누구나 부담 없이 의견 표출 가능

**인증 방법**
- Privy 지갑 연동 (온체인 자산 자동 확인)
- 자격증/라이선스 제출 (프로)

## 신뢰 점수 시스템

### 점수 구성

총 신뢰 점수 = 기본 점수 + 활동 점수 + 예측 점수 + 인증 점수

| 항목 | 점수 범위 | 설명 |
|------|-----------|------|
| 기본 점수 | 0-100 | 가입 기간, 레벨 |
| 활동 점수 | 0-200 | 글, 댓글, 좋아요 |
| 예측 점수 | 0-300 | 난이도 가중 점수, 참여 횟수 |
| 인증 점수 | 0-200 | 스킨 인 더 게임, 프로 인증 |

**최대 점수**: 800점

### 신뢰 등급

| 등급 | 점수 | 표시 |
|------|------|------|
| 신규 | 0-99 | 회색 |
| 일반 | 100-299 | 청색 |
| 신뢰 | 300-499 | 녹색 |
| 고신뢰 | 500-699 | 보라 |
| 최고 신뢰 | 700+ | 금색 |

### 점수 표시
- 프로필에 점수 및 등급 표시
- 글/댓글에 등급 아이콘 표시
- 마우스 오버 시 상세 점수

|------|-----------|

## 배지 우선순위

프로필에 표시되는 배지 순서

1. 특수 배지 (운영팀, 공식)
2. 예측가 배지 (최고 등급)
3. 인증 배지
4. 인플루언서 배지
5. 활동 배지

**표시 개수**
- 프로필: 전체 표시
- 글/댓글: 상위 3개
- 미니 프로필: 상위 1개

## FAQ

**Q: 배지는 어디서 확인하나요?**
A: 프로필 > 배지 탭에서 보유/미보유 배지를 확인할 수 있습니다.

**Q: 예측 점수는 어떻게 계산되나요?**
A: 단순 적중률이 아닌 **난이도 가중치** 시스템입니다. 남들이 틀릴 때 맞추면 높은 점수를 받습니다. 쉬운 예측만 해서는 배지를 얻기 어렵습니다.

**Q: 자산 표시를 꼭 해야 하나요?**
A: 아니요, **완전히 선택 사항**입니다. 기본값은 OFF이며, 원하는 분만 켜면 됩니다. 익명 글쓰기 모드도 있어요.

**Q: 스킨 인 더 게임 배지가 뭔가요?**
A: 글에서 언급한 종목을 **실제로 보유 중**일 때 표시됩니다. 보유 금액은 안 보이고, "이 사람은 진짜 들고 있음"만 표시해요.

**Q: 배지를 숨길 수 있나요?**
A: 설정 > 프로필 > 배지 표시에서 특정 배지를 숨길 수 있습니다. 글 작성 시 익명 모드를 쓰면 모든 배지가 숨겨집니다.

**Q: 신뢰 점수가 갑자기 떨어졌어요.**
A: 90일 동안의 활동만 반영되므로, 최근 활동이 줄면 점수가 하락할 수 있습니다.
