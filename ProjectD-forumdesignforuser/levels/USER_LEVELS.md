## 레벨업 조건 상세

### Lv.1 → Lv.2 (뉴비 → 루키)
- 경험치 100 이상
- 글 3개 이상 작성
- 제재 이력 없음

### Lv.2 → Lv.3 (루키 → 레귤러)
- 경험치 500 이상
- 글 10개 이상 작성
- 예측 5회 이상
- 가입 후 7일 경과

### Lv.3 → Lv.4 (레귤러 → 시니어)
- 경험치 2,000 이상
- 글 30개 이상 작성
- 예측 적중률 50% 이상 (20회 기준)
- 가입 후 30일 경과

### Lv.4 → Lv.5 (시니어 → 프로)
- 경험치 5,000 이상
- 글 100개 이상 작성
- 예측 적중률 55% 이상 (50회 기준)
- 가입 후 90일 경과

### Lv.5 → Lv.6 (프로 → 마스터)
- 경험치 15,000 이상
- 글 300개 이상 작성
- 예측 적중률 60% 이상 (100회 기준)
- 가입 후 180일 경과
- 커뮤니티 기여 활동

## 레벨 다운

### 다운 조건

| 사유 | 감소 | 비고 |
|------|------|------|
| 30일 미접속 | -100 경험치 | 월 1회 |
| 경고 누적 | -200 경험치 | 경고당 |
| 일시 정지 | -500 경험치 | 정지당 |
| 장기 정지 | -1,000 경험치 | 정지당 |

### 레벨 보호
- 레벨 1 미만으로 하락하지 않음
- 마스터 레벨은 90일 미활동 시에만 다운
- 프로 이상은 경고로 인한 다운 없음 (정지만 해당)

### 복구
- 활동 재개 시 경험치 재획득
- 제재 해제 후 일부 경험치 복구 가능

## 레벨 표시

### 표시 위치
- 프로필 페이지 (이름 옆)
- 글/댓글 작성자 정보
- 팔로워/팔로잉 목록
- 검색 결과

### 표시 형식
```
닉네임 🌳 Lv.3
```

### 경험치 바
- 프로필에서 다음 레벨까지 진행률 표시
- 필요 경험치 / 현재 경험치 표시

|------|------|

## FAQ

**Q: 레벨이 갑자기 떨어졌어요.**
A: 장기 미접속 또는 제재로 인한 경험치 감소일 수 있습니다. 마이페이지 > 활동 내역에서 확인하세요.

**Q: 경험치는 어디서 확인하나요?**
A: 프로필 > 레벨 탭에서 현재 경험치와 다음 레벨까지 필요한 경험치를 확인할 수 있습니다.

**Q: 글을 삭제하면 경험치도 차감되나요?**
A: 네, 삭제된 글에서 획득한 경험치는 차감됩니다. 단, 받은 좋아요 경험치는 유지됩니다.

**Q: 최대 레벨에 도달하면 경험치는 어떻게 되나요?**
A: 마스터 레벨 달성 후에도 경험치는 계속 누적됩니다. 향후 추가 레벨이나 시즌 순위에 반영됩니다.
