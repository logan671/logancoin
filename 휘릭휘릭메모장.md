# 휘릭휘릭메모장(휘발)

## 복붙용 명령어

### Step 1: 커밋+푸시 자동화 명령(alias) 등록
> `git c "YYMMDD HHMM"` 형태로 커밋하면 자동으로 push까지 실행되도록 alias를 등록.

```bash
git config --global alias.c '!f(){ git add . && git commit -m "$1"; }; f'
```

### Step 2: 자동 푸시 hook 실행 권한 확인
> 커밋 후 자동 푸시 hook이 활성인지 확인.

```bash
ls -l /Users/hwlee/Desktop/업무/codefolder/mac1codefolder/logancoin/.git/hooks/post-commit
```

### Step 3: 실제 사용 예시
> 커밋하면 자동으로 푸시됨.

```bash
cd /Users/hwlee/Desktop/업무/codefolder/mac1codefolder/logancoin && git c "260215 2246"
```
