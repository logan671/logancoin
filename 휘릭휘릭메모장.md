### Step 3: (선택) 관찰 로그 개수 확인
> SQLite에 관찰 로그가 쌓였는지 개수를 확인하는 명령어.

```bash
cd /Users/hwlee/Desktop/업무/codefolder/mac1codefolder/logancoin/ProjectG-15minpolybetter && python3 - <<'PY'
import sqlite3
from pathlib import Path
db_path = Path("data/polybot.db")
if not db_path.exists():
    print("DB not found:", db_path)
else:
    with sqlite3.connect(db_path) as conn:
        count = conn.execute("SELECT COUNT(*) FROM observations").fetchone()[0]
    print("observations:", count)
PY
```
