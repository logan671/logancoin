# 휘릭휘릭메모장(휘발)

## 복붙용 명령어

### Step 1: ProjectH 폴더에서 현재 Git 상태 확인
> 지금 폴더가 Git으로 연결돼 있는지, 원격 저장소가 붙어 있는지 확인하는 명령어.

```bash
cd ~/Desktop/기본/codefolder/claudecode && git status && git remote -v
```

### Step 2: (GitHub에서 새 저장소 만든 뒤) 원격 저장소 연결
> 내 컴퓨터 코드와 GitHub 새 저장소를 연결하는 명령어.

```bash
cd ~/Desktop/기본/codefolder/claudecode && (git remote set-url origin git@github.com:여기에네아이디/여기에저장소이름.git || git remote add origin git@github.com:여기에네아이디/여기에저장소이름.git) && git remote -v
```

### Step 3: 처음 업로드(커밋 + 푸시)
> 지금까지 만든 파일을 GitHub에 올리는 명령어.

```bash
cd ~/Desktop/기본/codefolder/claudecode && git add . && git commit -m "260212 4PM (Codex)" && git push -u origin main
```
