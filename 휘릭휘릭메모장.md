# 휘릭휘릭메모장(휘발)

## 복붙용 명령어

### Step 1: 커밋 (자동 푸시됨)
> 커밋하면 자동으로 GitHub에 푸시됨.

```bash
cd /Users/hwlee/Desktop/업무/codefolder/mac1codefolder/logancoin && git c "260215 2315"
```

### Step 2: 서버 업데이트
> 서버에서 최신 코드 받고 서비스 재시작.

```bash
cd ~/app && git pull && sudo systemctl restart projectg-observer
```

### Step 3: 이메일 리포트 환경변수 파일 생성
> 서버에 이메일 발송 설정을 저장.

```bash
sudo tee /etc/projectg-report.env >/dev/null <<'EOF'
REPORT_EMAIL_FROM=money0stack9@gmail.com
REPORT_EMAIL_TO=money0stack9@gmail.com
REPORT_SMTP_USER=money0stack9@gmail.com
REPORT_SMTP_PASSWORD=PASTE_APP_PASSWORD_HERE
EOF
```

### Step 4: 리포트 타이머 설치/활성화
> 1시간 주기로 리포트 실행, 첫 24시간 이후는 하루 1회만 발송.

```bash
sudo cp /home/ubuntu/app/ProjectG-15minpolybetter/deploy/projectg-report.service /etc/systemd/system/
sudo cp /home/ubuntu/app/ProjectG-15minpolybetter/deploy/projectg-report.timer /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now projectg-report.timer
```

### Step 5: 리포트 타이머 상태 확인
> 타이머가 활성화됐는지 확인.

```bash
systemctl status projectg-report.timer
```
